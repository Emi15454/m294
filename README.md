# m294

## Projektauftrag Portfolio

Diese Website wurde als digitales Portfolio mit rein eigener HTML-, CSS- und JavaScript-Implementierung umgesetzt. Die Inhalte werden über eine JSON-Datei gesteuert, damit Projekte flexibel erweitert werden können.

## Anforderungen erfüllt

- Mehrere Hauptseiten: Home, Projekte, Detailseite und Kontakt
- Responsives Grid-Layout für Projektübersicht und Inhalte
- Newsletter- und Kontaktformulare
- Projektfilter nach Kategorie
- Detailseite mit Projektbeschreibung, Bildern und Anfrageformular
- Merkliste über Web Storage API

## KI-Einsatz

- VS Code mit GitHub Copilot für Code-Generierung und Refactoring
- ChatGPT zur Strukturierung von Inhalten und Copywriting
- Beide Tools wurden genutzt, um Layoutideen, Komponentenstruktur und Inhalte effizient zu entwickeln

## Wireframes und Styleguide

- Wireframes für Mobile, Tablet und Desktop liegen als Planungsdateien im Repository vor
- Ein einfacher Styleguide ist in der Projektstruktur dokumentiert
