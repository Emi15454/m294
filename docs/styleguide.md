# Styleguide

## Farben

- Hintergrund: #0d0d0d
- Oberfläche: #1a1a1a
- Text: #f9f9f9
- Sekundärtext: #a1a1aa
- Akzent: #3b82f6

## Typografie

- Hauptfont: Inter, system-ui, sans-serif
- Überschriften: 700, tight line-height
- Fließtext: 400, klare Lesbarkeit

## Komponenten

- Buttons: rundungslose, klare Akzentfarbe
- Karten: abgesetzte Oberfläche mit leichter Schattenwirkung
- Formulare: kontrastreiche Eingabefelder mit Fokusfarbe
